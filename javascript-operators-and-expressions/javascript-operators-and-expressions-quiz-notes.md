# javascript-operators-and-expressions-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What data type is returned by an arithmetic operation?
A numerical value is returned by an arithmetic operation
- What is string concatenation?
String concatenation means to join two or more text strings into one string.
- What purpose(s) does the `+` plus operator serve in JavaScript?
The "+" is used to either add two numerical values together or to combine a string to another string.
- What data type is returned by comparing two values (`<`, `>`, `===`, etc)?
When returning with comparing two values, you return a Boolean.
- What does the `+=` "plus-equals" operator do?
The += operator adds the value on its right to the variable or property on its left, and assigns the result to the variable or property on its left.
